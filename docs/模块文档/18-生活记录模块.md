## 一、模块概述

### 1.1 模块定位

生活记录模块是幼儿园管理系统的核心业务模块之一，负责教师每日录入幼儿的生活状态数据，包括饮食、午睡、如厕、健康监测等方面。

### 1.2 使用角色

| 角色 | user_type | 权限说明 |
|:---|:---:|:---|
| 班主任 | 1 | 可进入，可录入，数据限本班 |
| 配班老师 | 1 | 可进入，可录入，数据限本班 |
| 保健医生 | 1 | 可进入，可录入，数据限本班 |
| 家长 | 2 | 不可进入 |

## 三、数据字典

### 3.1 生活记录表（busi_life_record）

| 字段名 | 中文名 | 字段类型 | 可空 | 默认值 | 说明 |
|:---|:---|:---|:---:|:---:|:---|
| id | 主键 | bigint | 否 | AUTO | 主键ID |
| student_id | 幼儿ID | bigint | 否 | | 关联幼儿 |
| class_id | 班级ID | bigint | 否 | | 所属班级 |
| record_date | 记录日期 | date | 否 | | 记录日期 |
| lunch_intake | 午餐摄入量 | tinyint | 是 | NULL | 1-全吃光 2-吃大半 3-吃小半 4-没胃口 |
| water_count | 喝水次数 | tinyint | 是 | NULL | 次数 |
| sleep_start_time | 入睡时间 | time | 是 | NULL | 入睡时间 |
| sleep_duration | 睡眠时长 | int | 是 | NULL | 时长（分钟） |
| sleep_quality | 睡眠质量 | tinyint | 是 | NULL | 1-深睡 2-浅睡 3-易醒 |
| toilet_count | 如厕次数 | tinyint | 是 | NULL | 次数 |
| toilet_abnormal | 异常情况 | varchar(255) | 是 | NULL | 异常描述 |
| morning_temp | 晨检体温 | decimal(4,1) | 是 | NULL | 体温（℃） |
| afternoon_temp | 午检体温 | decimal(4,1) | 是 | NULL | 体温（℃） |
| health_symptoms | 不适症状 | varchar(255) | 是 | NULL | 症状描述 |
| teacher_remark | 教师备注 | text | 是 | NULL | 备注 |
| creator_id | 创建人ID | bigint | 是 | NULL | 创建人 |
| create_time | 创建时间 | datetime | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | 更新时间 | datetime | 是 | CURRENT_TIMESTAMP | 更新时间 |
| is_deleted | 逻辑删除 | tinyint | 是 | 0 | 0-未删除 1-已删除 |

**索引**：
- PRIMARY KEY (`id`)
- INDEX `idx_student_date` (`student_id`, `record_date`)

## 四、功能说明

### 4.1 快速填充功能

#### 交互入口

在支持快速填充的列标题右侧，显示 ⚡ 图标按钮：

```
│ 🍽️ 午餐摄入量 [⚡] │ 💧 喝水次数 [⚡] │ ... │ 😴 睡眠质量 [⚡] │
```

#### 填充流程

1. 点击列头 ⚡ 按钮
2. 选择填充值（默认推荐值）
3. 勾选"仅填充空白单元格"
4. 确认填充

#### 默认填充值

| 字段 | 默认填充值 | 推荐理由 |
|:---|:---:|:---|
| 午餐摄入量 | 全吃光 | 正常情况 |
| 喝水次数 | 3 | 日常标准 |
| 睡眠质量 | 深睡 | 正常情况 |

### 4.2 保存机制

| 触发条件 | 动作 | 反馈 |
|:---|:---|:---|
| 输入框失焦 | 自动保存当前字段 | 单元格显示绿点 |
| 下拉选择变更 | 自动保存当前字段 | 单元格显示绿点 |
| 时间选择器确认 | 自动保存当前字段 | 单元格显示绿点 |
| 保存失败 | 显示错误 | 单元格显示红点 |

### 4.3 状态反馈

**单元格级别状态**：

| 状态 | 视觉表现 | 说明 |
|:---|:---|:---|
| 未修改 | 无标记 | 初始状态或未改动 |
| 修改中 | 单元格边框高亮 | 正在编辑 |
| 已保存 | 单元格右下角显示绿点 ● | 自动保存成功 |
| 保存失败 | 单元格显示红点 ● + 红色边框 | 需要重试 |

**页面级别状态（右上角）**：

| 状态 | 视觉表现 | 说明 |
|:---|:---|:---|
| 已同步 | 🟢 已同步 | 所有更改已保存 |
| 同步中 | 🟡 同步中... | 保存请求中 |
| 同步失败 | 🔴 同步失败 | 点击可重试 |

## 五、API 接口

### 5.1 查询生活记录列表

> **接口路径**：`GET /api/busi/life-record/page`

| 参数名 | 类型 | 必填 | 说明 |
|:---|:---|:---:|:---|
| classId | Long | 是 | 班级ID |
| date | String | 是 | 记录日期（yyyy-MM-dd） |

### 5.2 保存生活记录

> **接口路径**：`POST /api/busi/life-record/save`

| 参数名 | 类型 | 必填 | 说明 |
|:---|:---|:---:|:---|
| studentId | Long | 是 | 幼儿ID |
| classId | Long | 是 | 班级ID |
| recordDate | String | 是 | 记录日期 |
| lunchIntake | Integer | 否 | 午餐摄入量 |
| waterCount | Integer | 否 | 喝水次数 |
| sleepStartTime | String | 否 | 入睡时间 |
| sleepDuration | Integer | 否 | 睡眠时长 |
| sleepQuality | Integer | 否 | 睡眠质量 |
| toiletCount | Integer | 否 | 如厕次数 |
| toiletAbnormal | String | 否 | 异常情况 |
| morningTemp | BigDecimal | 否 | 晨检体温 |
| afternoonTemp | BigDecimal | 否 | 午检体温 |
| healthSymptoms | String | 否 | 不适症状 |
| teacherRemark | String | 否 | 教师备注 |

### 5.3 批量保存生活记录

> **接口路径**：`POST /api/busi/life-record/batch-save`

适用于快速填充功能，批量保存同一列的多个幼儿记录。
