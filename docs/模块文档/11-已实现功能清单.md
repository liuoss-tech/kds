### 5.2 权限树结构

```
├── 首页看板 (id=4) [隐藏，默认拥有]
```

### 5.3 权限分配特性

| 特性 | 说明 |
|------|------|
| 首页权限隐藏 | 权限分配界面不显示首页权限 |
| 首页默认拥有 | 所有角色自动拥有首页访问权限 |
| 半选状态处理 | 前端合并全选和半选节点提交 |
| 先删后增策略 | 权限分配时先删除原有权限再插入新权限 |
| 动态菜单 | 根据用户角色动态渲染左侧导航菜单 |

## 六、数据隔离功能（2026-03-18 新增）

### 6.1 实现组件

| 组件 | 路径 | 说明 |
|------|------|------|
| LoginHelper | `com.kds.common.context.LoginHelper` | 用户上下文工具类，获取当前登录用户信息 |
| LoginHelper.isAdmin() | - | 判断是否为管理员（SUPER_ADMIN/PRINCIPAL角色） |
| LoginHelper.isTeacher() | - | 判断是否为教职工（userType=1） |
| LoginHelper.isParent() | - | 判断是否为家长（userType=2） |
| CustomDataPermissionHandler | `com.kds.config.CustomDataPermissionHandler` | 数据权限处理器，动态注入过滤条件 |
| MybatisPlusConfig | `com.kds.config.MybatisPlusConfig` | 集成数据权限拦截器 |
| BusiStudentParent | `com.kds.model.entity.BusiStudentParent` | 幼儿-家长绑定关系实体 |
| BusiStudentParentMapper | `com.kds.mapper.BusiStudentParentMapper` | 查询家长绑定的孩子ID |

### 6.2 数据范围定义

| data_scope | 角色 | 说明 |
|:---:|:---|:---|
| 1 | SUPER_ADMIN/PRINCIPAL | 全园数据 - 可查看所有数据 |
| 2 | HEAD_TEACHER/TEACHER | 本班数据 - 可查看任教的班级 |
| 4 | PARENT | 仅限绑定关系 - 可查看与自己孩子相关的数据 |

### 6.3 隔离效果

| 角色 | 班级列表 | 幼儿列表 | 请假记录 |
|------|----------|----------|----------|
| 管理员/园长 | 查看全部 | 查看全部 | 查看全部 |
| 班主任/老师 | 任教班级 | 本班幼儿 | 本班请假 |
| 家长 | - | 自己孩子 | 自己孩子请假 |

### 6.4 过滤逻辑

```
老师 (data_scope=2):
  SQL自动注入: WHERE class_id IN (分配的班级ID列表)
  
家长 (data_scope=4):
  SQL自动注入: WHERE id IN (绑定孩子ID列表)
  
管理员/园长 (data_scope=1):
  无过滤条件
```

### 6.5 不参与隔离的表

系统表不进行数据权限过滤：
- sys_user, sys_role, sys_permission
- sys_user_role, sys_role_permission
- busi_teacher_class, busi_student_parent
