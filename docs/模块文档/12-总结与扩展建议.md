## 五、开发规范

### 5.1 后端规范

- Controller：负责接收请求、参数校验、返回响应
- Service：负责业务逻辑处理
- Mapper：负责数据库操作
- Entity：对应数据库表
- VO：用于前端展示
- DTO：用于数据传输和参数校验

### 5.2 前端规范

- 组件命名：使用大驼峰命名
- 样式命名：使用 BEM 命名规范
- API 请求：统一封装在 request.js
- 状态管理：使用 Vuex 或 Pinia
