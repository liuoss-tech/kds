# 食谱管理模块

> **文档版本**：1.0.0
> **创建日期**：2026-03-20

---

## 一、模块概述

食谱管理模块是幼儿园管理系统的核心业务模块之一，负责教职工为班级规划和记录每日食谱信息。

### 1.1 模块范围

| 类型 | 说明 |
|:---|:---|
| **本期范围** | 教职工端食谱管理页面（新增、编辑、查看） |
| **后续扩展** | 家长端食谱查看功能（不在本期范围） |

### 1.2 使用角色

| 角色 | user_type | 权限说明 |
|:---|:---:|:---|
| 园长/管理员 | 1 | 可查看/编辑所有班级食谱 |
| 班主任 | 1 | 可查看/编辑所任班级食谱 |
| 配班老师 | 1 | 可查看/编辑所任班级食谱 |
| 保健医生 | 1 | 可查看/编辑所任班级食谱 |
| 家长 | 2 | 不可进入 |

### 1.3 数据范围

根据 `busi_teacher_class` 表，教职工属于哪些班级，就能在哪些班级下管理食谱。

---

## 二、功能清单

### 2.1 核心功能

| 功能 | 说明 |
|:---|:---|
| 班级选择 | 可切换查看/编辑不同班级的食谱 |
| 日期导航 | 支持左右箭头切换上一天/下一天，日历选择器跳转任意日期 |
| 今日快速跳转 | 一键回到当日 |
| 餐次编辑 | 早餐、早点、午餐、午点、晚餐（可选） |
| 显示/隐藏 | 早点、晚餐默认折叠，点击展开/收起 |
| 保存食谱 | 统一保存按钮，必填项校验 |

### 2.2 餐次说明

| 餐次 | 默认状态 | 必填 |
|:---|:---:|:---:|
| 早餐 | 展开 | 是 |
| 早点 | 折叠 | 否 |
| 午餐 | 展开 | 是 |
| 午点 | 展开 | 否 |
| 晚餐 | 折叠 | 否 |

## 三、API 接口

### 3.1 获取食谱详情

| 项目 | 内容 |
|:---|:---|
| **接口路径** | `/api/busi/recipe` |
| **请求方法** | `GET` |
| **认证要求** | 是（教职工用户） |

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|:---|:---:|:---:|:---|
| classId | Long | 是 | 班级ID |
| date | String | 是 | 食谱日期（yyyy-MM-dd） |

**响应示例**：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "id": 1,
    "classId": 1,
    "className": "大一班",
    "targetDate": "2026-03-20",
    "breakfast": "鸡蛋、牛奶、面包",
    "morningSnack": "水果",
    "lunch": "米饭、红烧肉、青菜、番茄蛋汤",
    "afternoonSnack": "酸奶",
    "dinner": "",
    "publisherId": 10,
    "publisherName": "张老师"
  }
}
```

### 3.2 保存食谱

| 项目 | 内容 |
|:---|:---|
| **接口路径** | `/api/busi/recipe` |
| **请求方法** | `POST` |
| **认证要求** | 是（教职工用户） |

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|:---|:---:|:---:|:---|
| classId | Long | 是 | 班级ID |
| targetDate | String | 是 | 食谱日期 |
| breakfast | String | 是 | 早餐内容 |
| morningSnack | String | 否 | 早点内容 |
| lunch | String | 是 | 午餐内容 |
| afternoonSnack | String | 否 | 午点内容 |
| dinner | String | 否 | 晚餐内容 |

**响应示例**：

```json
{
  "code": 200,
  "message": "食谱保存成功",
  "data": null
}
```

## 四、页面设计

### 4.1 页面布局

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│  🍽️ 食谱管理                                                                    │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│     [班级 ▼] 大一班     [◀ 2026-03-20 周四 ▶]  [📅]  [今日]                    │
│                                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────────┐    │
│  │ 🌅 早餐                                                                   │    │
│  │ ┌───────────────────────────────────────────────────────────────────┐  │    │
│  │ │ 请输入早餐内容...                                                      │  │    │
│  │ └───────────────────────────────────────────────────────────────────┘  │    │
│  └─────────────────────────────────────────────────────────────────────────┘    │
│                                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────────┐    │
│  │ 🍎 早点                                    [显示/隐藏]                    │    │
│  │ ...折叠内容...                                                          │    │
│  └─────────────────────────────────────────────────────────────────────────┘    │
│                                                                                 │
│  ...午餐、午点卡片...                                                           │
│                                                                                 │
│  ...晚餐（可选）卡片...                                                        │
│                                                                                 │
│                        [ 💾 保存食谱 ]                                          │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 4.2 交互说明

1. **页面初始化**：获取当前用户所在班级列表，默认选中第一个班级，默认日期为当天，加载当天食谱
2. **班级切换**：切换班级后，加载新班级的当天食谱
3. **日期切换**：切换日期后，加载新日期的食谱（无数据则显示空白表单）
4. **保存逻辑**：校验早餐、午餐必填，保存后显示成功提示

## 五、数据字典

### 5.1 食谱表（busi_recipe）

| 字段名 | 中文名 | 必填 | 说明 |
|:---|:---|:---:|:---|
| id | 主键ID | - | 存在则更新，不存在则新增 |
| class_id | 班级ID | 是 | 关联 busi_class |
| target_date | 食谱日期 | 是 | 唯一约束：(class_id, target_date) |
| breakfast | 早餐 | 是 | 255字符以内 |
| morning_snack | 早点 | 否 | 255字符以内 |
| lunch | 午餐 | 是 | 255字符以内 |
| afternoon_snack | 午点 | 否 | 255字符以内 |
| dinner | 晚餐 | 否 | 255字符以内 |
| publisher_id | 发布人ID | 是 | 当前登录用户ID |
| create_time | 创建时间 | - | 自动填充 |
| update_time | 更新时间 | - | 自动更新 |
| is_deleted | 逻辑删除 | - | 默认0 |
