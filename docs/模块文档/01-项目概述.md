# 幼儿园家校管理系统 - 项目概述

> **文档版本**：1.0.0
> **更新时间**：2026-03-15

---

## 一、项目背景

本项目为**幼儿园家校联系系统**（KDS Kindergarten System），旨在为幼儿园提供一套完整的家校沟通与园所管理解决方案。系统采用前后端分离架构，后端基于 Spring Boot 构建，前端采用 Vue 3 + Element Plus 开发。

---

## 二、技术栈总览

| 层级 | 技术选型 | 版本 |
|------|----------|------|
| **后端框架** | Spring Boot | 2.7.18 |
| **持久层框架** | MyBatis Plus | 3.5.5 |
| **认证鉴权** | Sa-Token | 1.37.0 |
| **数据库** | MySQL | 8.0+ |
| **缓存** | Redis | - |
| **前端框架** | Vue 3 | 3.4+ |
| **UI组件库** | Element Plus | 2.5+ |
| **路由** | Vue Router | 4.x |
| **构建工具** | Vite | 5.x |

## 四、模块清单

| 序号 | 模块名称 | 说明 |
|:---:|:---|:---|
| 1 | 用户认证与鉴权 | 登录、退出、Token认证 |
| 2 | 用户管理 | 教职工和家长用户管理 |
| 3 | 角色管理 | 角色CRUD、权限分配 |
| 4 | 班级管理 | 幼儿园班级管理 |
| 5 | 前端路由与导航 | Vue Router配置、路由守卫 |
| 6 | HTTP请求封装 | Axios拦截器封装 |
