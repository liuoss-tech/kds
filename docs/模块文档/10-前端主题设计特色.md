# 前端主题设计特色

> **文档版本**：1.0.0
> **更新时间**：2026-03-15

---

## 一、设计理念

项目前端采用了**温馨可爱的幼儿园主题设计**，旨在为用户营造一个友好、温暖的使用体验。

---

## 二、登录页面设计

### 2.1 视觉设计

| 元素 | 描述 |
|------|------|
| **背景** | 渐变背景（暖色调：淡黄→浅粉） |
| **装饰** | 漂浮云朵动画 |
| **元素** | 彩虹装饰 |
| **动画** | 气泡漂浮效果 |
| **Logo** | 卡通emoji |

### 2.2 功能特点

- 表单验证（账号、密码必填）
- 记住账号功能（localStorage）
- 登录成功后存储 Token 及用户信息到 localStorage
- 带动画的幼儿园主题 UI 设计

---

## 三、管理页面设计

### 3.1 视觉设计

| 元素 | 描述 |
|------|------|
| **按钮** | 粉色系渐变按钮 |
| **卡片** | 圆角卡片设计 |
| **图标** | 卡通图标装饰 |
| **动画** | 流畅的交互动画 |

### 3.2 布局结构

- **侧边栏**：显示菜单导航
- **头部**：显示用户信息、退出按钮
- **主内容区**：显示页面内容

## 五、组件设计

### 5.1 按钮

```css
/* 渐变按钮 */
.btn-primary {
  background: linear-gradient(135deg, #FF69B4, #FFB6C1);
  border-radius: 20px;
  color: white;
}
```

### 5.2 卡片

```css
/* 圆角卡片 */
.card {
  border-radius: 16px;
  box-shadow: 0 4px 12px rgba(255, 105, 180, 0.1);
}
```

### 5.3 表格

```css
/* 表格样式 */
.el-table {
  border-radius: 12px;
  overflow: hidden;
}
```

## 六、动画效果

### 6.1 过渡动画

```css
/* 页面切换动画 */
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s;
}

.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
```

### 6.2 按钮悬停

```css
/* 按钮悬停效果 */
.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 6px 16px rgba(255, 105, 180, 0.3);
}
```

## 七、响应式设计

### 7.1 断点设置

| 断点 | 屏幕宽度 | 布局 |
|------|----------|------|
| xs | < 768px | 单列布局 |
| sm | 768px - 992px | 双列布局 |
| md | 992px - 1200px | 三列布局 |
| lg | > 1200px | 四列布局 |

### 7.2 适配策略

- 使用 Element Plus 的响应式组件
- 灵活使用栅格系统 (el-row, el-col)
- 图片使用 max-width: 100%
