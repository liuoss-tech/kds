## 三、路由配置

### 3.1 路由结构

```javascript
const routes = [
  {
    path: '/login',
    name: 'Login',
    component: () => import('@/views/Login.vue')
  },
  {
    path: '/',
    component: Layout,
    redirect: '/dashboard',
    children: [
      { path: 'dashboard', name: 'Dashboard', component: () => import('@/views/Dashboard.vue') },
      { path: 'system/user', name: 'SystemUser', component: () => import('@/views/system/User.vue') },
      { path: 'system/role', name: 'RoleManagement', component: () => import('@/views/system/Role.vue') }
    ]
  }
]
```

### 3.2 路由说明

| 路径 | 组件 | 说明 |
|------|------|------|
| `/login` | Login.vue | 登录页 |
| `/` | Layout | 布局组件（包含侧边栏和头部） |
| `/dashboard` | Dashboard.vue | 首页看板 |
| `/system/user` | system/User.vue | 用户管理 |
| `/system/role` | system/Role.vue | 角色管理 |

## 四、路由守卫

### 4.1 鉴权控制

```javascript
// 路由守卫 - 鉴权控制
router.beforeEach((to, from, next) => {
  const token = localStorage.getItem('kds_token');

  if (to.path === '/login') {
    // 已登录用户访问登录页，跳转到首页
    token ? next({ path: '/' }) : next();
  } else {
    // 未登录用户访问其他页面，跳转到登录页
    token ? next() : next(`/login?redirect=${to.path}`);
  }
});
```

### 4.2 守卫执行流程

```
1. 用户访问页面
2. 检查目标路径是否为登录页
   ├── 是：检查是否已登录
   │         ├── 已登录 → 重定向到首页
   │         └── 未登录 → 放行
   └── 否：检查是否有Token
             ├── 有Token → 放行
             └── 无Token → 跳转到登录页（带redirect参数）
```

## 五、布局结构

### 5.1 Layout 组件

布局组件包含以下部分：
- **侧边栏 (Sidebar)**：显示菜单导航
- **头部 (Header)**：显示用户信息、退出按钮
- **主内容区 (Main)**：显示页面内容

### 5.2 菜单渲染

菜单根据用户角色动态渲染，调用后端权限接口获取菜单数据。

## 六、路由懒加载

项目使用 Vue Router 的懒加载功能：

```javascript
// 非懒加载
import User from '@/views/system/User.vue'
{ path: 'system/user', component: User }

// 懒加载（推荐）
{ path: 'system/user', component: () => import('@/views/system/User.vue') }
```

**懒加载优势：**
- 减少首屏加载时间
- 按需加载页面资源
- 提升用户体验

## 七、路由元信息

### 7.1 Meta 字段

可以通过路由的 meta 字段设置额外信息：

```javascript
{
  path: '/system/user',
  component: () => import('@/views/system/User.vue'),
  meta: {
    title: '用户管理',
    icon: 'User',
    requiresAuth: true,
    roles: ['SUPER_ADMIN', 'PRINCIPAL']
  }
}
```

### 7.2 常用 Meta 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `title` | String | 页面标题 |
| `icon` | String | 菜单图标 |
| `requiresAuth` | Boolean | 是否需要登录 |
| `roles` | Array | 允许访问的角色 |

## 八、页面跳转方式

### 8.1 编程式导航

```javascript
// 跳转页面
this.$router.push('/dashboard')
this.$router.push({ path: '/dashboard' })
this.$router.push({ name: 'Dashboard' })

// 带查询参数
this.$router.push({ path: '/dashboard', query: { id: 1 } })

// 带路由参数
this.$router.push({ name: 'User', params: { id: 1 } })

// 替换页面（不记录历史）
this.$router.replace('/dashboard')

// 前进/后退
this.$router.go(1)   // 前进
this.$router.go(-1)  // 后退
```

### 8.2 声明式导航

```html
<router-link to="/dashboard">跳转</router-link>
<router-link :to="{ path: '/dashboard' }">跳转</router-link>
```
