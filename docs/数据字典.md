- **数据库名称**： kindergarten_management
- **字符集**： utf8mb4

## 三、详细表结构

### 1. sys_user（系统用户表）

| 序号 | 字段名 | 字段类型 | 可空 | 默认值 | 说明 |
|:---:|:---|:---|:---:|:---:|:---|
| 1 | id | bigint | 否 | AUTO_INCREMENT | 主键ID |
| 2 | username | varchar(50) | 否 | | 登录账号(建议用手机号) |
| 3 | password | varchar(100) | 否 | | 登录密码(需哈希加密) |
| 4 | real_name | varchar(50) | 否 | | 真实姓名 |
| 5 | phone | varchar(20) | 是 | NULL | 联系电话 |
| 6 | user_type | tinyint | 否 | | 用户类型：1-内部教职工 2-家长 |
| 7 | status | tinyint | 是 | 1 | 账号状态：0-禁用 1-正常 |
| 8 | create_time | datetime | 是 | CURRENT_TIMESTAMP | 创建时间 |
| 9 | update_time | datetime | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
| 10 | is_deleted | tinyint | 是 | 0 | 逻辑删除：0-未删除 1-已删除 |

**索引**：
- PRIMARY KEY (`id`)
- UNIQUE KEY `uk_username` (`username`)
