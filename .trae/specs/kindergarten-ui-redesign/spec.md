# 幼儿园家园联系系统 UI/UX 改造规范

## 背景
当前登录界面和Layout布局虽然功能完整，但风格较为通用。需要针对幼儿园场景进行温馨童趣的UI/UX改造，提升用户体验和品牌辨识度。

## 改造内容

### 1. 登录界面改造
- **背景设计**：从深色渐变改为温暖的奶油色渐变，搭配柔和的云朵/星星/彩虹装饰元素
- **登录卡片**：采用更圆润的卡通风格，带有淡淡的卡通阴影
- **输入框**：增加柔和的边框发光效果，聚焦时有温暖的渐变边框
- **登录按钮**：改为彩虹渐变，带有糖果般的质感和弹性动画
- **Logo区域**：使用更具幼儿园特色的emoji组合

### 2. Layout布局改造
- **侧边栏**：改为温暖的奶油白 + 淡粉渐变，添加手绘风格的装饰线条
- **Logo区域**：设计可爱的小房子+花朵组合图标
- **菜单项**：增加柔和的悬停动画，选中状态更活泼
- **顶部导航栏**：添加温馨的装饰元素，用户头像区域增加互动效果
- **背景装饰**：主内容区添加流动的云朵、星星等童趣元素

## 影响范围
- 受影响文件：
  - `frontend/src/views/Login.vue` - 登录页面
  - `frontend/src/layout/Index.vue` - 主布局

## 新增需求

### 需求：温馨奶油童话风格登录页
系统应提供温馨可爱的登录界面，让家长和老师有温暖的体验

#### 场景：用户访问登录页
- **当** 用户打开登录页面
- **应看到** 奶油色渐变背景、童趣装饰元素、圆润可爱的登录卡片

#### 场景：用户输入账号密码
- **当** 用户聚焦输入框
- **应看到** 温暖的边框发光效果，图标颜色变化

#### 场景：用户点击登录
- **当** 用户点击登录按钮
- **应看到** 按钮有弹性缩放效果，Loading时有可爱的动画

### 需求：温馨童趣主布局
系统应提供温馨可爱的主界面布局

#### 场景：用户查看侧边栏
- **当** 用户看到侧边栏
- **应看到** 奶油白+淡粉渐变，可爱的小房子Logo，流畅的菜单动画

#### 场景：用户与菜单交互
- **当** 用户悬停或点击菜单项
- **应看到** 柔和的动画过渡，选中项有活泼的渐变背景

## 修改需求
### 需求：现有登录界面
保留原有的表单验证逻辑、记住账号、密码显示切换功能，仅改造视觉风格

### 需求：现有Layout布局
保留原有的菜单权限过滤、用户信息显示、退出登录功能，仅改造视觉风格
